# Netflix Clone

[https://netflixclone-ad9a3.firebaseapp.com/](https://neflixclonesite.netlify.app)

## Run it locally
1. git clone https://github.com/amitpanwar789/Netflix-Clone
2. cd Netflic-Clone
3. npm install
